## Installation

### Browser
* Use any internet browser preffered updated version eg: chrome, mozila, safari
* Make sure you have internet connection

### Rails
  * clone the repo
  * Use command bundle install

### Posgress
  * rails db:create
  * rails db:seed

### React
  * [clone this repo](https://github.com/Emkayint/swahili-client)
  * npm install client

## Usage / BDD

